# 我的第一份 Markdown 文件

這是用來練習 Pandoc 轉換的範例文件。

## 章節一：基礎介紹

Markdown 是一種**輕量級標記語言**，讓你可以用純文字格式撰寫文件。

### 特點

- 簡單易學
- 純文字格式
- 可轉換成多種格式

### 程式碼

行內程式碼：`pandoc sample.md -o sample.docx`

程式碼區塊：

```bash
# 安裝 Pandoc
winget install pandoc

# 轉換文件
pandoc input.md -o output.docx
```

| 功能 | 說明 |
|------|------|

| 粗體 | 使用 ** 包圍 |
| 列表 | 使用 - 或數字 |
