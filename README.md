# Antigravity + Pandoc AI 整理術

完整的 10 Part 課程，結合 Google Antigravity (AI IDE) 與 Pandoc 文件轉換工具，學習如何用 AI 提升文件處理效率。

## 🎯 立即開始

### 👉 [點此開始觀看簡報](https://chatgpt3a01.github.io/Antigravity-Pandoc-AI/)

---

## 安裝教學

### Google Antigravity 安裝

[點此下載安裝教學影片](https://drive.google.com/file/d/1CZd48Mxd-isH-bnJg0ydS13zwQ9N-uFu/view?usp=sharing)

### Pandoc 安裝

- **官方下載**: https://pandoc.org/installing.html
- **Windows**: 下載 `.msi` 安裝檔執行即可

---

## 課程大綱

| Part | 主題 | 內容 |
|------|------|------|
| 01 | 認識 Antigravity | 功能總覽、與其他工具比較 |
| 02 | 環境安裝與設定 | 下載安裝、介面導覽、AI 模型選擇 |
| 03 | Pandoc 安裝與基礎轉換 | 安裝 Pandoc、第一次轉換、常見錯誤 |
| 04 | 批次轉換與自動化 | 批次處理、自動化腳本、AI 產生腳本 |
| 05 | 進階格式轉換 | HTML 樣式、PDF 輸出、自訂模板 |
| 06 | AI 輔助文件處理 | 整理結構、生成摘要、校正潤飾 |
| 07 | 實戰專案 | 會議記錄、技術文件、批次報告 |
| 08 | 進階技巧與優化 | 效能優化、錯誤處理、流程整合 |
| 09 | 疑難排解 | 問題診斷、環境設定、AI 協助 |
| 10 | 總結與延伸 | 課程複習、學習資源、個人工作流 |

---

## 使用方式

1. 開啟各 Part 資料夾中的 `.html` 簡報檔案
2. 使用瀏覽器直接開啟即可觀看
3. 簡報支援鍵盤 ← → 翻頁

## 簡報功能

- **首頁**: 跳回第一頁
- **全螢幕**: 切換全螢幕模式
- **FB**: 連結 Facebook
- **YT**: 連結 YouTube 頻道
