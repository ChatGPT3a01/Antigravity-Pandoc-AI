# 進階轉換指令速查表

## HTML 轉換

```powershell
# 基本 HTML（片段）
pandoc input.md -o output.html

# 完整 HTML 文件
pandoc input.md -o output.html --standalone

# 套用 CSS
pandoc input.md -o output.html --standalone --css=style.css

# 嵌入所有資源（單一檔案）
pandoc input.md -o output.html --standalone --css=style.css --embed-resources

# 加入目錄
pandoc input.md -o output.html --standalone --toc --toc-depth=3

# 程式碼高亮
pandoc input.md -o output.html --standalone --highlight-style=pygments
```

## PDF 轉換（使用 wkhtmltopdf）

```powershell
# 基本轉換（先轉 HTML）
pandoc input.md -o temp.html --standalone --css=style.css
wkhtmltopdf temp.html output.pdf

# 設定頁面
wkhtmltopdf --page-size A4 --orientation Portrait input.html output.pdf

# 設定邊距
wkhtmltopdf --margin-top 20 --margin-bottom 25 --margin-left 25 --margin-right 25 input.html output.pdf

# 加入頁碼
wkhtmltopdf --footer-center "第 [page] 頁，共 [topage] 頁" --footer-font-size 10 input.html output.pdf

# 加入頁首
wkhtmltopdf --header-center "文件標題" --header-line input.html output.pdf

# 加入目錄
wkhtmltopdf toc input.html output.pdf

# 加入封面
wkhtmltopdf cover cover.html content.html output.pdf
```

## 使用模板

```powershell
# Word 參考模板
pandoc input.md -o output.docx --reference-doc=template.docx

# 產生預設 Word 模板
pandoc -o reference.docx --print-default-data-file reference.docx

# HTML 模板
pandoc input.md -o output.html --template=template.html

# 匯出預設 HTML 模板
pandoc -o template.html --print-default-template=html5
```
